# Orlando Pirates Fan Experience App

A mobile-responsive web app built with Expo and React Native Web, designed for live deployment to Vercel.

## 🎨 Brand Colors

- **Primary Teal**: `#1B9B9E` - CTAs, active states
- **Navy Blue**: `#0A1E3D` - Headers, navigation
- **Orange**: `#FF6B35` - Time-sensitive items
- **Gold**: `#C9A961` - Premium features, badges

## 🚀 Quick Start

### Install Dependencies

```bash
cd /Users/missioncontrol/SeamlessMarketplace/EventBrella/orlandoPirates
npm install --legacy-peer-deps
```

### Start Development Server

```bash
npm start
```

### Test Web Build

```bash
npm run vercel-build
```

This creates a `dist/` folder ready for deployment.

### Deploy to Vercel

The app is configured for automatic Vercel deployment:

1. Push to GitHub/GitLab
2. Connect repository to Vercel
3. Vercel will auto-detect the configuration and deploy

Or use Vercel CLI:
```bash
vercel --prod
```

## 📱 Features

- **Digital Ticketing Hub** - QR codes, match schedule
- **Merch of the Day** - Daily deals with QR codes
- **Fan Polls & Engagement** - Live voting, leaderboard
- **Purchase Tracking** - History, spending insights
- **Fan Journey Tracking** - Entry/exit placeholders

## 🏗️ Project Structure

- `src/screens/` - 5 main screens
- `src/components/` - Reusable UI components
- `src/constants/` - Colors, spacing, typography
- `src/data/` - Mock data files
- `src/navigation/` - Bottom tab navigation
