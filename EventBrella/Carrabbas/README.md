# Carrabbas Fan Experience App

A mobile-responsive web app built with Expo and React Native Web.

## 🎨 Brand Colors

- **Primary Teal**: `#1B9B9E` - CTAs, active states
- **Navy Blue**: `#0A1E3D` - Headers, navigation
- **Orange**: `#FF6B35` - Time-sensitive items
- **Gold**: `#C9A961` - Premium features, badges

## 🚀 Quick Start

### Install Dependencies

```bash
cd /Users/missioncontrol/SeamlessMarketplace/EventBrella/Carrabbas
npm install
```
### Start Development Server

```bash
npm start
```

### Build for Web

```bash
npm run build:web
```

This creates a `dist/` folder ready for deployment to any static host.

## 📱 Features

- **Digital Ticketing Hub** - QR codes, match schedule
- **Merch of the Day** - Daily deals with QR codes
- **Fan Polls & Engagement** - Live voting, leaderboard
- **Purchase Tracking** - History, spending insights
- **Fan Journey Tracking** - Entry/exit placeholders

## 🏗️ Project Structure

- `src/screens/` - 5 main screens
- `src/components/` - Reusable UI components
- `src/constants/` - Colors, spacing, typography
- `src/data/` - Mock data files
- `src/navigation/` - Bottom tab navigation

## 📦 Tech Stack

- Expo SDK 54
- React Native 0.81.5
- React Navigation 7
- React Native Web
- Expo Linear Gradient
- React Native QR Code SVG
